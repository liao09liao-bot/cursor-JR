## 🌐 部署網站

### 選項 1：Vercel 部署（推薦）

1. **前往 Vercel**
   - 訪問：https://vercel.com
   - 使用 GitHub 帳號登入

2. **匯入專案**
   - 點擊 "New Project"
   - 選擇 `cursor-JR` 儲存庫
   - Vercel 會自動偵測 Vite 專案

3. **部署設定**
   - Framework Preset: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`

4. **部署**
   - 點擊 "Deploy"
   - 等待部署完成
   - 獲得網站網址（例如：`cursor-jr.vercel.app`）

### 選項 2：Netlify 部署

1. **前往 Netlify**
   - 訪問：https://netlify.com
   - 使用 GitHub 帳號登入

2. **匯入專案**
   - 點擊 "Add new site" → "Import an existing project"
   - 選擇 `cursor-JR` 儲存庫

3. **建置設定**
   - Build command: `npm run build`
   - Publish directory: `dist`

4. **部署**
   - 點擊 "Deploy site"
   - 等待部署完成

### 選項 3：GitHub Pages

1. **安裝 gh-pages**
   ```bash
   npm install --save-dev gh-pages
   ```

2. **在 package.json 中添加腳本**
   ```json
   {
     "scripts": {
       "deploy": "npm run build && gh-pages -d dist"
     }
   }
   ```

3. **部署**
   ```bash
   npm run deploy
   ```
