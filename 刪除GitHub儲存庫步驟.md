# 如何刪除 GitHub 儲存庫

## ⚠️ 警告
**刪除儲存庫是永久性的操作，無法復原！** 請確認您真的要刪除這個儲存庫。

## 刪除步驟

### 方法一：透過 GitHub 網站（推薦）

1. **進入儲存庫頁面**
   - 點擊「遊標-JR」儲存庫進入詳情頁面

2. **進入設定頁面**
   - 點擊儲存庫上方的「設定」（Settings）分頁
   - 或直接前往：`https://github.com/liao09liao-bot/遊標-JR/settings`

3. **找到危險區域（Danger Zone）**
   - 滾動到頁面最下方
   - 找到紅色邊框的「Danger Zone」（危險區域）區塊

4. **刪除儲存庫**
   - 點擊「Delete this repository」（刪除此儲存庫）按鈕
   - 或「刪除此儲存庫」按鈕

5. **確認刪除**
   - 輸入儲存庫名稱 `liao09liao-bot/遊標-JR` 或 `遊標-JR` 來確認
   - 閱讀警告訊息
   - 點擊「I understand the consequences, delete this repository」（我了解後果，刪除此儲存庫）

6. **完成**
   - 儲存庫將被永久刪除
   - 您會被重新導向到個人資料頁面

### 方法二：透過 GitHub Desktop（如果已連接）

1. 在 GitHub Desktop 中右鍵點擊儲存庫
2. 選擇「View on GitHub」（在 GitHub 上查看）
3. 然後按照「方法一」的步驟進行

## 注意事項

### 刪除前的備份建議
- ✅ 確認本地已經有完整的程式碼備份
- ✅ 確認所有重要的檔案都已下載
- ✅ 如果有協作者，通知他們先備份

### 刪除後的影響
- ❌ 儲存庫的所有程式碼將被永久刪除
- ❌ 所有 Issues、Pull Requests、Wiki 等都會被刪除
- ❌ 所有協作者的存取權限將被移除
- ❌ 無法復原（除非有備份）

### 如果只想隱藏儲存庫
如果您不想刪除，而是想讓它變成私有：
1. 進入儲存庫的「設定」（Settings）
2. 滾動到「Danger Zone」
3. 點擊「Change visibility」（變更可見性）
4. 選擇「Make private」（設為私有）

## 快速連結
