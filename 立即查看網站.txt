========================================
如何立即查看網站
========================================

最簡單的方法：

1. 雙擊執行「啟動網站.bat」
   
   或者

2. 在終端機執行：
   npm run dev

3. 等待幾秒鐘，瀏覽器會自動開啟
   如果沒有自動開啟，訪問：http://localhost:5173

4. 您會看到完整的網站，包括：
   - 首頁 Hero 區塊（大標題和動畫）
   - 產品展示區（6個產品卡片）
   - 品牌故事區
   - 聯絡表單
   - 購物車功能
   - 登入註冊功能

5. 停止伺服器：在終端機按 Ctrl + C

========================================
網站特色
========================================

- 工業金屬風格設計
- 銀灰、石墨藍配色
- 玻璃擬態效果
- 流暢動畫
- 響應式設計（手機、平板、桌面）

========================================
如果無法啟動
========================================

1. 確認 Node.js 已安裝
   下載：https://nodejs.org/

2. 安裝依賴：
   npm install

3. 重新啟動：
   npm run dev
