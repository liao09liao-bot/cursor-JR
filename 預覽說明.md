# 如何預覽網站

## 步驟 1：安裝 Node.js

如果您的電腦還沒有安裝 Node.js，請先安裝：

1. 前往 [Node.js 官網](https://nodejs.org/)
2. 下載 LTS 版本（建議版本）
3. 執行安裝程式並完成安裝
4. 重新開啟終端機/命令提示字元

## 步驟 2：安裝專案依賴

開啟終端機（Terminal / PowerShell / CMD），進入專案目錄，然後執行：

```bash
npm install
```

這個指令會安裝所有必要的套件（React、Tailwind、Framer Motion 等）

## 步驟 3：啟動開發伺服器

安裝完成後，執行：

```bash
npm run dev
```

## 步驟 4：開啟瀏覽器預覽

當您看到類似以下的訊息時：

```
  VITE v5.0.0  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

這表示開發伺服器已經啟動！

**請在瀏覽器中開啟：`http://localhost:5173`**

## 常見問題

### 如果出現 "npm 不是內部或外部命令"

這表示 Node.js 沒有正確安裝，請重新安裝 Node.js 並確保選擇「自動添加到 PATH」選項。

### 如果端口 5173 已被佔用

Vite 會自動使用下一個可用端口（如 5174），請查看終端機輸出的實際網址。

### 如何停止開發伺服器

在終端機中按 `Ctrl + C` 即可停止。

## 其他有用的指令

### 建置生產版本
```bash
npm run build
```

### 預覽生產版本
```bash
npm run preview
```

## 快速預覽（如果您已安裝 Node.js）

直接在專案目錄執行以下兩個指令：

```bash
npm install
npm run dev
```

然後開啟瀏覽器訪問 `http://localhost:5173`
