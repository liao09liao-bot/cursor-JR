# 如何查看網站

## 🚀 方法一：本地預覽（最簡單，推薦）

### 步驟 1：啟動開發伺服器

**方式 A：使用腳本（最簡單）**
1. 雙擊執行 `啟動網站.bat`
2. 等待幾秒鐘，瀏覽器會自動開啟
3. 如果沒有自動開啟，手動訪問：http://localhost:5173

**方式 B：使用命令列**
```bash
npm run dev
```

### 步驟 2：查看網站

- 網站會自動在瀏覽器中開啟
- 如果沒有自動開啟，訪問：http://localhost:5173
- 您會看到完整的網站，包括：
  - ✅ Hero 區塊（首頁大圖）
  - ✅ 產品展示區
  - ✅ 品牌故事
  - ✅ 聯絡表單
  - ✅ 購物車功能
  - ✅ 登入註冊功能

### 步驟 3：停止伺服器

在終端機中按 `Ctrl + C` 即可停止。

## 📱 網站功能展示

啟動網站後，您可以測試以下功能：

### 1. 首頁（Hero 區）
- 大標題「JR Industrial Mounts」
- 標語「工業級電視架與洗衣機底座專家」
- 「查看商品」按鈕（點擊會滾動到產品區）

### 2. 產品展示區
- 6 個產品卡片
- 每個產品顯示：名稱、規格、價格
- 「加入購物車」和「詢價」按鈕
- 滑鼠懸停有動畫效果

### 3. 品牌故事區
- 左側：品牌介紹文字
- 右側：工廠圖片（佔位圖）
- 4 個特色標籤

### 4. 聯絡表單
- 姓名、公司、聯絡方式、留言欄位
- 提交後顯示成功訊息

### 5. 購物車功能
- 點擊右上角購物車圖示
- 查看購物車內容
- 調整數量、移除商品
- 計算總價

### 6. 登入註冊
- 點擊右上角「登入」按鈕
- 可以登入或註冊
- 管理員可以管理商品

## 🚨 如果網站無法啟動

### 問題 1：Node.js 未安裝

**解決方法**：
1. 前往 https://nodejs.org/
2. 下載並安裝 LTS 版本
3. 重新開啟終端機
4. 執行 `npm install`
5. 執行 `npm run dev`

### 問題 2：依賴未安裝

**解決方法**：
```bash
npm install
```

### 問題 3：端口被佔用

**解決方法**：
- Vite 會自動使用下一個可用端口
- 查看終端機輸出的實際網址

### 問題 4：出現錯誤訊息

**解決方法**：
1. 檢查終端機的錯誤訊息
2. 確認所有檔案都已正確保存
3. 嘗試刪除 `node_modules` 和 `package-lock.json`
4. 重新執行 `npm install`
