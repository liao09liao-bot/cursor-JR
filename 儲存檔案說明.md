## 💾 儲存方式

### 方式一：自動儲存（IDE）
- 在編輯器中，檔案會自動儲存
- 或按 `Ctrl + S` 手動儲存

### 方式二：提交到 Git（推薦）
使用以下方式提交變更到 GitHub：

**使用腳本（最簡單）**：
1. 雙擊執行 `儲存並提交.bat`
2. 腳本會自動：
   - 添加所有變更
   - 提交變更
   - 推送到 GitHub

**使用命令列**：
```bash
git add .
git commit -m "Fix GitHub Pages configuration and add website launch scripts"
git push origin main
```

**使用 GitHub Desktop**：
1. 開啟 GitHub Desktop
2. 在「Changes」分頁查看變更
3. 填寫提交訊息
4. 點擊「Commit to main」
5. 點擊「Push origin」
